# Install

## Local Project Install

```bash
mkdir -p .claude/skills
git clone https://github.com/your-org/shimei-lab.git .claude/skills/shimei-lab
```

## Global Install

```bash
git clone https://github.com/your-org/shimei-lab.git ~/.claude/skills/shimei-lab
```

## Optional Dependencies

```bash
pip3 install -r requirements.txt
```
